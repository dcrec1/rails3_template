Whenever you want to create a new project, just do:

<pre><code>rails new app_name -d postgresql -S -T -m https://github.com/dcrec1/rails6_template/raw/master/template.rb</code></pre>

Change <tt>app_name</tt> for your project name. You can configure what you need and run <tt>rake db:migrate</tt> to get started.
